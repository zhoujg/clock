## [未发布] - 2026-06-14

### 新增功能

#### ❤️ 音乐收藏功能

为 BGM 音乐播放器添加了完整的收藏功能，让用户可以收藏喜欢的音乐并快速访问。

**主要功能：**

- ❤️ 一键收藏当前播放的音乐
- 📋 专属收藏列表视图，独立于正常播放列表
- 🎵 从收藏列表直接播放音乐
- 🗑️ 灵活的收藏管理（移除、取消收藏）
- 💾 本地持久化存储（localStorage）
- 🎨 精美的 UI 设计和交互动画
- 📊 实时显示收藏数量
- 💬 收藏操作的即时提示反馈

**技术特性：**

- 使用心形图标表示收藏状态（粉红色=已收藏，白色=未收藏）
- 收藏按钮带有心跳动画效果
- 收藏列表支持移除操作（鼠标悬停显示删除按钮）
- 通知消息使用滑入/滑出动画
- 完整的事件系统支持状态同步

**使用方式：**

1. 播放音乐时，点击播放控制区域最左侧的心形按钮收藏
2. 点击"我的收藏"按钮查看收藏列表
3. 在收藏列表中点击音乐即可播放
4. 鼠标悬停在收藏项上可以看到移除按钮

详见：`MUSIC_FAVORITES_GUIDE.md`

## [未发布] - 2026-06-13

### 新增功能

#### 🎵 BGM 音乐播放器

添加了全功能的背景音乐播放器，让用户在使用时钟时可以播放舒缓的背景音乐。

**主要功能：**

- ✨ 播放/暂停、上一曲/下一曲控制
- 🔁 单曲循环模式
- 📊 进度条显示和快速跳转
- 🔊 音量调节（0-100%）
- 📋 音乐列表管理
- 💾 自动保存播放设置
- 📱 响应式设计，支持移动端
- 🎨 现代化 UI，可展开/收起

**技术实现：**

- 新增文件：
  - `scripts/bgmPlayer.js` - 播放器核心逻辑
  - `styles/bgmPlayer.css` - 播放器样式
  - `assets/bgm/` - 音乐文件目录
  - `BGM_PLAYER_GUIDE.md` - 使用指南

- 集成到主应用：
  - 在 `app.js` 中初始化播放器管理器
  - 在 `index.html` 中添加播放器 UI
  - 在 `storage.js` 中保存播放器设置

**支持的音频格式：**
- MP3 (推荐)
- WAV
- OGG
- M4A

**使用方法：**

1. 将音乐文件放入 `assets/bgm/` 目录
2. 在 `scripts/bgmPlayer.js` 的 `loadMusicList()` 方法中配置音乐信息
3. 刷新页面，点击右下角音乐图标开始使用

详细使用说明请查看 `BGM_PLAYER_GUIDE.md`。
