# 每日三个故事 - 快速开始 🚀

## 🎯 5分钟快速体验

### 步骤 1：启动服务器

打开终端，运行：

```bash
cd /Users/zhoujingen/Documents/BangSuite/clock
./start-server.sh
```

你会看到：
```
🚀 启动本地服务器...
📍 访问地址: http://localhost:8000
⏹️  按 Ctrl+C 停止服务器
```

### 步骤 2：打开浏览器

在浏览器中访问：**http://localhost:8000**

### 步骤 3：体验新功能

1. **找到 🎯 按钮**
   - 位置：左上角第一个按钮
   - 徽章显示：`0/3`（红色）

2. **打开面板**
   - 点击 🎯 按钮
   - 左侧滑入"每日三个故事"面板

3. **填写第一个故事**
   ```
   标题：完成项目文档
   描述：今天我要完成这个项目文档，因为它关系到团队的进度，体现了我对工作的责任心
   价值观：点击 "💼 责任"
   ```

4. **填写第二个故事**
   ```
   标题：学习 React Hooks
   描述：花一小时学习 React Hooks，这是我技术成长路径中的重要一步
   价值观：点击 "📚 学习"
   ```

5. **填写第三个故事**
   ```
   标题：陪孩子玩耍
   描述：下班后陪孩子玩半小时积木，这是我作为父亲的重要时光
   价值观：点击 "👨‍👩‍👧 家庭"
   ```

6. **完成故事**
   - 完成一个故事后，勾选它的复选框 ✓
   - 观察完成动画和进度变化
   - 完成所有三个，体验庆祝动画 🎉

7. **查看统计**
   - 点击底部"查看统计"按钮
   - 查看你的完成情况和价值观分布

## 📱 移动端体验

如果想在手机上测试：

1. **确保手机和电脑在同一WiFi**

2. **查找电脑IP地址**
   ```bash
   ifconfig | grep "inet " | grep -v 127.0.0.1
   ```

3. **在手机浏览器访问**
   ```
   http://[你的电脑IP]:8000
   ```
   例如：`http://192.168.1.100:8000`

## 🔍 调试模式

### 查看控制台

1. 打开浏览器开发者工具（F12）
2. 查看 Console 标签
3. 查找任何错误信息

在控制台执行：

## ⚠️ 常见问题

### Q: 点击 🎯 按钮没反应？
A: 
1. 检查浏览器控制台是否有错误
2. 确认 `dailyStories.js` 和 `dailyStories.css` 已正确加载
3. 刷新页面重试

### Q: 数据没有保存？
A: 
1. 检查浏览器是否启用了 localStorage
2. 检查是否处于隐私模式
3. 查看控制台是否有存储错误

### Q: 样式显示不正常？
A: 
1. 硬刷新页面（Ctrl+Shift+R 或 Cmd+Shift+R）
2. 清除浏览器缓存
3. 检查 CSS 文件是否正确加载

### Q: 在移动端无法访问？
A: 
1. 确认手机和电脑在同一网络
2. 检查防火墙设置
3. 尝试使用电脑的实际IP地址而不是localhost
