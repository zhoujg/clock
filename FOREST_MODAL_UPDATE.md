# 🌲 森林模态框更新

## 📱 更新原因

由于应用将转换为 APP，打开新标签页的方式不适合移动应用。现在将森林展示改为**模态弹窗**形式，与成就系统保持一致。

---

## 🎨 新实现方式

### 之前（Web 方式）
```javascript
// 点击按钮打开新标签页
window.open('forest.html', '_blank');
```

### 现在（APP 方式）✅
```javascript
// 点击按钮显示全屏模态框
this.showForestModal();
```

## 🎨 视觉效果

### 打开动画
```
背景遮罩：
opacity: 0 → 1
duration: 0.3s

内容区域：
opacity: 0 → 1
transform: scale(0.9) → scale(1)
duration: 0.3s
```

### 树木动画
```
每棵树依次出现：
animation-delay: index * 0.02s (最多 2s)

淡入上浮：
opacity: 0 → 1
translateY: 20px → 0
duration: 0.5s
```

### 悬停效果
```
树木卡片悬停：
- 向上浮动 5px
- 放大 1.05 倍
- 显示绿色光晕
- 边框变亮
```

### 关闭动画
```
反向播放打开动画：
opacity: 1 → 0
transform: scale(1) → scale(0.9)
duration: 0.3s
```
