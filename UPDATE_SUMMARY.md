## 🚀 使用指南

### 开始种树
```
1. 点击 🍅 番茄钟按钮
2. 选择时长（15/25/45/60 分钟）
3. 点击播放按钮
4. 右下角出现树苗 🌱
5. 观察树苗成长
6. 完成后获得树木 🌿
```

### 查看森林
```
1. 点击 🌲 森林按钮（左上角）
2. 查看统计和最近记录
3. 点击「查看完整森林」
4. 全屏模态框展开
5. 欣赏你的森林
6. 点击背景或关闭按钮退出
```
