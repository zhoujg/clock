🚀 如何使用

  1. 添加音乐文件
    将音乐文件（MP3、WAV、OGG、M4A）放入 assets/bgm/ 目录
  2. 配置音乐列表
    编辑 scripts/bgmPlayer.js，在 loadMusicList() 方法中添加： 
    this.musicList = [
    { name: '轻音乐1', file: 'assets/bgm/music1.mp3' },
    { name: '轻音乐2', file: 'assets/bgm/music2.mp3' }
];
  3. 打开应用
    刷新页面，点击右下角的音乐图标 🎵

⚠️ 注意事项

需要手动添加音乐文件

由于浏览器安全限制，无法自动扫描本地文件。你需要：

  1. 手动将音乐文件放入 assets/bgm/ 目录
  2. 在 bgmPlayer.js 中配置文件列表

详细说明请查看 assets/bgm/README.md
