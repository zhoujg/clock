# 每日三个故事 - 功能说明

## 📖 功能简介

"每日三个故事"是一个基于目标管理理念的每日规划系统。通过把每天的重要活动与你的期望和价值关联起来，让看似平淡的活动变得更有意义。

## 💡 核心理念

### 三个故事的力量

人与人最大的区别在于如何使用时间。你的每一天过得是否有意义，完全取决于自己。通过"三个故事"，你可以：

1. **找到当天真正重要的三个结果**
   - 不是"要做的事"，而是"想要的结果"
   - 例如：割草、完成项目计划、享受一顿好午餐

2. **用故事赋予情绪**
   - 把枯燥的任务变成有意义的故事
   - 让你能够主动、积极地面对每一天

3. **与价值观联系**
   - 把故事与你的价值体系联系起来
   - 例如：责任、学习、健康、家庭、成长

## 🎯 如何使用

### 1. 打开面板

点击左上角的 🎯 图标，打开"每日三个故事"侧边栏。

### 2. 制定今日目标

每天早晨，思考并填写三个你今天真正想要完成的事情：

**示例：**

**故事 1：完成项目文档**
- 标题：完成项目文档
- 故事：今天我要完成这个项目文档，因为它关系到团队的进度，体现了我对工作的责任心
- 价值观：💼 责任

**故事 2：学习新技术**
- 标题：学习 React Hooks
- 故事：花一小时学习 React Hooks，这是我技术成长路径中的重要一步
- 价值观：📚 学习

**故事 3：陪孩子玩耍**
- 标题：陪孩子搭积木
- 故事：下班后陪孩子玩半小时积木，这是我作为父亲的重要时光
- 价值观：👨‍👩‍👧 家庭

### 3. 执行与完成

- 完成一个故事后，勾选复选框
- 系统会记录完成时间
- 完成所有三个故事后，会有特殊庆祝动画 🎉

### 4. 查看统计

点击底部"查看统计"按钮，可以看到：
- 📊 记录天数
- ✅ 完成天数
- 📈 完成率
- 💎 价值观分布
- 📅 最近7天趋势

- 顶部进度条显示今日完成情况
- 徽章显示 `已完成/总数`（如 2/3）
- 根据完成情况变色（红→黄→绿）

### 智能日期管理
- 每天自动刷新为新的一天
- 历史数据永久保存
- 支持查看历史记录

### 完成庆祝
- 单个故事完成有闪光动画
- 完成全部三个故事时弹出庆祝页面
- 正向激励，增强成就感

### 数据统计
- 累计记录天数
- 完成率统计
- 价值观分布分析
- 最近7天趋势图

## 🎓 使用技巧

### 技巧 1：SMART 原则
- **Specific**：具体明确（不是"学习"，而是"学习React Hooks"）
- **Measurable**：可衡量（完成项目文档的第3章）
- **Achievable**：可实现（一天能完成）
- **Relevant**：与价值观相关
- **Time-bound**：今天完成

### 技巧 2：平衡原则
三个故事最好涵盖不同领域：
- 1个工作相关
- 1个学习成长
- 1个生活/健康/家庭

### 技巧 3：避免"待办清单陷阱"
- ❌ 不好：回复邮件、买菜、洗衣服
- ✅ 好：完成重要项目、学习新技能、高质量陪伴家人

### 技巧 4：反思与调整
每晚问自己：
- 这三件事是否今天我真正想做的？
- 哪些做得好？哪些可以改进？
- 明天如何调整？

## ❓ 常见问题

**Q: 为什么是三个，不是五个或十个？**
A: 三个是经过验证的最佳数量，既有挑战性，又不会让人overwhelmed。专注于三个重要事项，而不是十个琐碎任务。

**Q: 一定要每天都填吗？**
A: 不强制，但建议养成习惯。即使周末也可以有三个故事，比如"好好休息"也是重要的结果。

**Q: 可以修改已完成的故事吗？**
A: 已完成的故事会锁定，避免误修改。如需调整，可以取消勾选后再编辑。

**Q: 数据会丢失吗？**
A: 数据保存在本地 localStorage，除非清除浏览器数据，否则不会丢失。未来会支持云端备份。

**Q: 可以在手机上使用吗？**
A: 完全支持移动端，响应式设计，操作流畅。
