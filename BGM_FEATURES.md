# BGM 音乐播放器 - 功能说明

## 🎯 核心特性

### 1. 自动扫描音乐文件 ⭐

**无需手动修改代码！** 播放器会自动从以下来源加载音乐：

#### 方法 1: music-list.json（推荐）

运行脚本自动生成音乐索引：

```bash
./scripts/generate-music-list.sh
# 或
node scripts/generate-music-list.js
```

**工作原理：**
- 扫描 `assets/bgm/` 目录
- 识别所有支持的音乐文件
- 生成 `music-list.json` 索引文件
- 播放器从索引加载音乐列表

#### 方法 2: Capacitor 文件系统（Android/iOS）

在移动应用中，播放器会直接使用 Capacitor API 读取文件系统：

```javascript
// 自动调用（无需手动配置）
const { Filesystem } = window.Capacitor.Plugins;
const files = await Filesystem.readdir({
    path: 'public/assets/bgm',
    directory: 'APPLICATION'
});
```

#### 方法 3: 文件探测（备用）

如果上述方法都不可用，播放器会尝试探测常见的文件名：
- `music1.mp3`, `music2.mp3`, ...
- `bgm1.mp3`, `bgm2.mp3`, ...
- `piano.mp3`, `rain.mp3`, `lofi.mp3`, ...

## 🎵 支持的音频格式

| 格式 | 扩展名 | 兼容性 | 推荐 |
|------|--------|--------|------|
| MP3 | `.mp3` | ⭐⭐⭐⭐⭐ | ✅ 推荐 |
| WAV | `.wav` | ⭐⭐⭐⭐ | 文件大 |
| OGG | `.ogg` | ⭐⭐⭐⭐ | 开源格式 |
| M4A | `.m4a` | ⭐⭐⭐⭐ | Apple 格式 |
| FLAC | `.flac` | ⭐⭐⭐ | 无损音质 |
| AAC | `.aac` | ⭐⭐⭐ | 高品质 |

**建议：** 使用 MP3 格式，兼容性最好，文件大小适中。

## 🎮 播放器功能

### 基础控制

| 功能 | 说明 | 图标 |
|------|------|------|
| 播放/暂停 | 点击中央大按钮 | ▶️ / ⏸ |
| 上一曲 | 播放前一首音乐 | ⏮️ |
| 下一曲 | 播放下一首音乐 | ⏭️ |
| 单曲循环 | 重复播放当前曲目 | 🔁 |

### 进度控制

- **进度条显示**：实时显示播放进度
- **时间显示**：当前时间 / 总时长
- **快速跳转**：点击进度条任意位置跳转

### 音量控制

- **音量滑块**：0-100% 可调
- **默认音量**：50%
- **自动保存**：下次打开恢复音量设置

### 界面控制

- **展开播放器**：点击右下角音乐图标 🎵
- **收起播放器**：点击面板右上角向上箭头 ⬆️
- **响应式设计**：自动适配移动端和桌面端

## 📝 智能文件名识别

播放器会自动从文件名提取音乐标题：

| 文件名 | 显示标题 |
|--------|----------|
| `calm_piano.mp3` | Calm Piano |
| `nature-sounds.mp3` | Nature Sounds |
| `lofi_study_01.mp3` | Lofi Study 01 |
| `Focus Music.mp3` | Focus Music |
| `rain_forest_ASMR.mp3` | Rain Forest Asmr |

**转换规则：**
1. 移除文件扩展名
2. 将 `_` 和 `-` 替换为空格
3. 每个单词首字母大写

## 🔧 高级功能

### 自动播放下一曲

播放完当前音乐后：
- **未开启循环**：自动播放下一曲
- **开启循环**：重复播放当前曲目
- **播放到最后一曲**：从第一曲开始循环

### 设置持久化

自动保存以下设置：
- ✅ 当前曲目索引
- ✅ 音量大小
- ✅ 循环模式状态

**注意：** 不会自动播放上次的音乐（遵守浏览器自动播放策略）

### 播放列表排序

音乐列表自动按名称字母顺序排序，方便查找。

## 📊 性能优化

### 音频预加载

播放器使用 `preload="auto"` 优化用户体验：
- 选择音乐时立即加载
- 减少播放延迟
- 智能缓存管理

### 错误处理

- 文件加载失败：显示错误提示
- 格式不支持：自动跳过
- 网络错误：提示用户检查连接

### 内存管理

- 一次只加载一个音频文件
- 切换音乐时释放之前的资源
- 避免内存泄漏

## 💡 使用场景

### 学习/工作专注

推荐音乐：
- Lo-fi / Chill Beats
- 轻音乐 / 纯音乐

- 自然音效（雨声、海浪、森林）
