1. `.story-modal-field-half`：允许 flex 子项缩小
2. `.story-modal-field input[type="date"], .story-modal-field input[type="time"]`：允许输入框自身缩小

## 23:37 — 每日故事插件安装时闪烁修复

## 23:45 — 启动闪烁问题进一步修复

**问题**：用户反馈"启动还是这样"，启动时仍然有闪烁。

**根因分析**：
1. `_ensureCSS()` 只是注入 `<link>` 标签但没有等待 CSS 真正加载完成，导致 `createUI()` 在 CSS 未加载时运行
2. 切换按钮（toggleBtn）创建后立即添加到 body，在 CSS 加载前会短暂显示在未样式化的位置

**修复**：
1. **`plugin.js` `_ensureCSS()`**：改为返回 Promise，监听 `link.onload` 和 `link.onerror`，确保 CSS 真正加载完成后再继续
2. **`plugin.js` `onActivate()`**：添加 `await _ensureCSS()` 等待 CSS 加载
3. **`dailyStories-class.js` `createUI()`**：为 toggleBtn 添加内联隐藏样式（`opacity=0` 和 `visibility=hidden`），然后使用双重 `requestAnimationFrame` 在 CSS 生效后再显示按钮
