# 2026-06-15 工作日志

## Safari 时间卡片缩小修复
- 根因：flip 库 `data-layout="horizontal fit"` 中的 `fit` 触发 JS 动态字体大小计算，Safari 亚像素渲染导致每秒累积缩小
- 修复：移除 `fit`，CSS `font-size: 15vw` 已足够响应式
- 修改：`index.html`, `android/.../public/index.html`

## iPad 滴答声间隔不均修复
- 根因：`tickSound.js` 用 HTML5 `<Audio>` 元素，iOS Safari `play()` 延迟不稳定
- 修复：改用 Web Audio API（预解码 + `AudioBufferSourceNode.start(0)`，延迟 <5ms）
- 修改：`scripts/tickSound.js`, `android/.../public/scripts/tickSound.js`
