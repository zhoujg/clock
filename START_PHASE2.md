## 💡 使用技巧

### 技巧 1：合理设定故事
```
✓ 好的故事：
  "完成项目文档第3章（具体）"
  "学习React Hooks（明确）"
  "健身1小时（可衡量）"

✗ 避免：
  "工作"（太宽泛）
  "学习"（不具体）
  "运动"（不可衡量）
```

### 技巧 2：番茄钟策略
```
大任务（2小时+）：
  → 4-5个番茄钟

中等任务（1小时）：
  → 2-3个番茄钟

小任务（30分钟）：
  → 1-2个番茄钟
```

### 技巧 3：价值观平衡
```
查看统计面板 → 价值观分布：

如果失衡：
- 💼 责任 70% → 工作太多，需要调整
- 💪 健康 5%  → 增加运动时间
- 👨‍👩‍👧 家庭 10% → 多陪伴家人
```

## 🐛 问题排查

### 问题 1：关联按钮不显示
**解决**：确保已填写故事标题

### 问题 2：数据没有更新
**解决**：
1. 确认通过"关联按钮"开始
2. 等待番茄钟完成
3. 刷新页面查看

### 问题 3：成就没有解锁
**解决**：
1. 打开控制台检查错误
2. 确认已完成故事
3. 查看成就面板是否已显示

### 问题 4：统计数据不准确
**解决**：
```javascript
// 控制台执行
const data = JSON.parse(localStorage.getItem('dailyStories'));
console.log(data);
// 检查 pomodoroCount 和 timeSpent 字段
```
