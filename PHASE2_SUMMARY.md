### 1. 系统架构

```
┌─────────────────────────────────────────────┐
│              App (app.js)                   │
│         应用主控制器和系统集成              │
└─────────────────────────────────────────────┘
                       │
        ┌──────────────┼──────────────┐
        │              │              │
        ▼              ▼              ▼
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│ DailyStories │ │  Pomodoro    │ │ Achievement  │
│   每日故事   │ │   番茄钟     │ │   成就系统   │
└──────────────┘ └──────────────┘ └──────────────┘
        │              │              │
        └──────────────┼──────────────┘
                       │
                       ▼
                ┌──────────────┐
                │   Forest     │
                │  森林系统    │
                └──────────────┘
```

### 2. 数据流

```
用户操作流程:
1. 设定故事 (DailyStories)
2. 点击关联番茄钟 (DailyStories → Pomodoro)
3. 完成番茄钟 (Pomodoro → DailyStories)
4. 更新故事数据 (DailyStories)
5. 完成故事 (DailyStories → Achievement)
6. 解锁成就 (Achievement)
7. 种植树木 (Pomodoro → Forest)

数据同步:
DailyStories.onPomodoroComplete(minutes)
  ↓ 更新 pomodoroCount, timeSpent
Achievement.checkStoriesAchievements(stats)
  ↓ 检查并解锁成就
  ↓ 添加经验值
Forest (未来扩展)
  ↓ 种植特殊树木
```
