## 💡 使用建议

### 制定有效的故事
1. **具体明确**："完成项目文档第3章" 比 "工作" 更好
2. **可实现**：确保今天能完成
3. **有意义**：与你的价值观相关
4. **平衡**：工作 + 学习 + 生活

### 培养习惯
1. **早晨规划**：每天早上设定三个故事
2. **晚上回顾**：检查完成情况，反思改进
3. **保持简单**：不要过度规划，三个足够
4. **持续追踪**：查看统计，观察自己的成长

## 🎯 核心理念

> **人与人最大的区别在于如何使用时间。**
> 
> 你的每一天过得是否有意义，完全取决于自己。
> 
> 通过把每天的活动与你的期望和价值观联系起来，
> 
> 让看似平淡的活动变得有意义。

## 📱 移动端访问

在手机上测试：

1. 查看电脑IP：`ifconfig | grep "inet "`
2. 手机浏览器访问：`http://[IP地址]:8000`
3. 例如：`http://192.168.1.100:8000`

## ⚠️ 注意事项

1. **数据存储**：使用浏览器 localStorage，清除缓存会丢失数据
2. **隐私模式**：隐私/无痕模式下可能无法保存数据
3. **跨设备**：数据不会自动同步到其他设备（未来可能支持云端）
4. **备份**：建议定期截图或导出重要数据（导出功能即将推出）

## 🐛 遇到问题？

### 检查控制台
按 F12 打开开发者工具，查看 Console 是否有错误

### 查看数据
在控制台执行：
```javascript
JSON.parse(localStorage.getItem('dailyStories'))
```

### 清除重置
在控制台执行：
```javascript
localStorage.removeItem('dailyStories')
location.reload()
```
