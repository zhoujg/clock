## 构建步骤详解

### 第一步：打开 Android Studio

```bash
npm run android:open
```

### 第二步：等待 Gradle 同步

首次打开项目时，Android Studio 会自动下载依赖（5-10分钟）

### 第三步：构建 APK

在 Android Studio 菜单栏：
1. 点击 `Build`
2. 选择 `Build Bundle(s) / APK(s)`
3. 点击 `Build APK(s)`
4. 等待构建完成（2-5分钟）

### 第四步：获取 APK

构建完成后，点击通知中的 "locate" 链接

APK 位置：
```
android/app/build/outputs/apk/debug/app-debug.apk
```

## 安装到手机

### 直接安装（推荐）

1. 连接手机到电脑（USB 线）
2. 在手机上启用"开发者选项"和"USB 调试"
3. 在 Android Studio 中点击绿色 "Run" 按钮
4. 选择你的设备
5. 应用自动安装并启动

### 传输 APK 安装

1. 将 `app-debug.apk` 传输到手机
2. 在手机上打开文件管理器
3. 点击 APK 文件安装
4. 允许"未知来源"安装（如果需要）

## 常见问题

### Q1: Android Studio 打不开？
**A**: 确保已下载并安装 Android Studio

### Q2: 应用显示空白？
**A**: 
1. 检查网络连接
2. 确认 https://neihou.cn/clock/ 能在浏览器访问
3. 在 Chrome 打开 `chrome://inspect` 查看错误

### Q3: 构建失败？
**A**: 
1. 等待 Gradle 完全同步完成
2. 点击 `Build` > `Clean Project`
3. 再次尝试构建

### Q4: 如何调试？
**A**: 
1. 在设备上运行应用
2. 在电脑 Chrome 打开：`chrome://inspect`
3. 找到应用并点击 "inspect"
