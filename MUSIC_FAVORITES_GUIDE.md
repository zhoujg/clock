# 🎵 音乐收藏功能使用指南

## 功能概述

现在你可以在播放音乐时收藏喜欢的曲目，并随时从收藏列表中快速播放它们。

## 主要功能

### 1. 收藏音乐 ❤️
- **位置**：在音乐播放器的控制按钮区域（最左侧的心形按钮）
- **操作**：
  - 播放音乐时，点击心形按钮即可收藏
  - 收藏成功后，按钮会变成粉红色心形，并有动画效果
  - 再次点击可以取消收藏
- **提示**：每次收藏或取消收藏都会在右上角显示通知消息

### 2. 查看收藏列表 📋
- **位置**：音量控制下方的"我的收藏"按钮
- **显示**：按钮上会显示当前收藏的音乐数量，如"我的收藏 (5)"
- **操作**：
  - 点击按钮切换到收藏列表视图
  - 收藏列表中的每首音乐前面都有❤️标记
  - 当前正在播放的收藏音乐会高亮显示
  - 再次点击按钮可以返回正常的音乐列表

### 3. 播放收藏的音乐 🎶
- **操作**：在收藏列表中，直接点击任意音乐即可播放
- **特点**：
  - 收藏的音乐会保留完整信息（曲名、艺术家、音频链接等）
  - 播放体验与正常播放完全一致
  - 支持所有播放控制（暂停、音量、进度等）

### 4. 管理收藏 🗑️
- **移除收藏**：
  - 方法1：在收藏列表中，鼠标悬停在音乐上会显示"×"按钮，点击即可移除
  - 方法2：播放该音乐后，点击心形收藏按钮取消收藏
- **自动保存**：所有收藏操作都会自动保存到浏览器本地存储

## 数据存储

- **存储位置**：浏览器 localStorage
- **存储键**：`musicFavorites`
- **数据格式**：JSON 数组，包含完整的音乐信息
- **持久化**：收藏的音乐会一直保存，直到手动移除或清除浏览器数据

## 技术实现

### 主要文件
1. **scripts/bgmPlayer.js** - 核心逻辑实现
   - `favoriteCurrentTrack()` - 收藏/取消收藏
   - `showFavoritesList()` - 显示收藏列表
   - `playFavoriteTrack()` - 播放收藏音乐
   - `removeFavorite()` - 移除收藏

2. **index.html** - UI 结构
   - 收藏按钮（心形图标）
   - 显示收藏列表按钮

3. **styles/bgmPlayer.css** - 样式定义
   - 收藏按钮样式和动画
   - 收藏列表样式
   - 移除按钮交互效果

4. **scripts/app.js** - 事件绑定
   - 收藏按钮点击事件
   - 切换收藏列表事件
   - 通知提示功能

## 使用场景

1. **发现好音乐**：浏览在线音乐时，遇到特别喜欢的立即收藏
2. **快速访问**：不用每次都搜索或翻找，直接从收藏列表播放
3. **个性化体验**：构建自己的专属音乐收藏，打造个人播放列表
4. **跨会话保存**：收藏会永久保存，下次打开页面时依然存在

## 交互提示

- ❤️ 粉红色心形 = 已收藏
- 🤍 白色心形 = 未收藏
- ✅ 绿色通知 = 收藏成功
- ℹ️ 蓝色通知 = 取消收藏

## 注意事项

1. 收藏的音乐链接来自 Jamendo API，确保网络连接正常
2. 如果音乐链接失效，可能需要重新搜索该音乐
3. 清除浏览器数据会删除所有收藏
4. 收藏数量没有限制，但建议控制在合理范围内以获得最佳体验
