- **播放音乐时** → 自动关闭滴答声

## ❌ 常见问题

### Q: 暂停后听不到滴答声？

**检查项**:
1. 播放音乐前滴答声是否开启？（如果关闭，暂停后也不会恢复）
2. 查看控制台是否显示 "滴答声已恢复并播放"
3. 检查浏览器音量是否开启
4. 尝试手动关闭再重新开启滴答声

### Q: 控制台显示错误？

**可能的错误**:
```
滴答声恢复播放失败: NotAllowedError
```

**解决方法**:
- 这是浏览器的自动播放限制
- 先手动点击页面任意位置
- 再测试播放-暂停功能

### Q: 有延迟？

**原因**:
- 音频文件未完全加载
- 浏览器性能问题

**解决**:
- 等待页面完全加载
- 检查网络连接
- 尝试刷新页面

## 🔧 技术说明

### 之前的实现（有延迟）
```javascript
resumeTickSound() {
    this.tickSoundManager.enabled = true; // 只设置标志
    // 需要等待时钟周期才播放（最多1秒延迟）
}
```

### 现在的实现（立即播放）
```javascript
resumeTickSound() {
    this.tickSoundManager.enabled = true; // 设置标志
    // 立即播放，不等待时钟周期
    this.tickSoundManager.audio.play();
}
```
