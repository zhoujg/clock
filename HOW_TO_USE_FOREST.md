## 📱 第一次使用

### 1. 打开应用
- 在浏览器中打开 `index.html`
- 你会看到漂亮的翻转时钟

### 2. 找到森林按钮
- 左上角，成就按钮 🏆 旁边
- 绿色的 🌲 图标
- 第一次使用时，徽章数字为 0

### 3. 开始第一棵树
1. 点击右上角 **🍅 番茄钟按钮**
2. 选择「工作」模式（默认 25 分钟）
3. 点击 **播放按钮 ▶️**
4. 🎉 开始种树了！

### 4. 观察成长
1. 点击 **🌲 森林按钮**
2. 看到面板显示：
   - 🌱 正在成长的树苗
   - 实时进度条
   - 剩余时间

### 5. 完成种植
- 等待倒计时结束
- 看到成功通知
- 徽章数字变成 1
- 恭喜！你种下了第一棵树 🎉

### 6. 查看森林
1. 再次点击 **🌲 森林按钮**
2. 点击「**🌳 查看完整森林**」
3. 新页面打开，展示你的森林
4. 第一棵树正在那里等着你！

## 💡 日常使用技巧

### 每天的流程
```
早上：
1. 打开时钟应用
2. 查看昨天的森林成果
3. 设定今天的目标（例如：种 3 棵树）

工作时：
4. 开始番茄钟
5. 专注工作/学习
6. 完成后查看新种的树
7. 休息一下
8. 继续下一个番茄钟

晚上：
9. 打开森林页面
10. 欣赏今天的成果
11. 统计学习时长
12. 为明天做计划
```

### 进阶技巧

#### 🎯 设定目标
```
每日目标：
- 新手：2-3 棵树（约 1 小时）
- 进阶：4-6 棵树（约 2 小时）
- 高手：8+ 棵树（约 3+ 小时）

每周目标：
- 种植 20+ 棵健康树木
- 保持 80%+ 成功率
- 尝试种植大树（45 分钟+）
```

#### 🌳 树木选择策略
```
早晨：
→ 小树苗（15分钟）热身

上午：
→ 小树（25分钟）标准番茄钟
→ 大树（45分钟）深度工作

下午：
→ 小树（25分钟）保持节奏
→ 参天大树（60分钟）挑战自我

晚上：
→ 小树苗（15分钟）轻松结束
```

#### 📊 提高成功率
```
1. 从短时长开始（15分钟）
2. 逐渐增加难度
3. 避免在疲劳时开始
4. 提前准备好工作内容
5. 关闭所有干扰源
6. 使用番茄工作法
7. 适当安排休息时间
```

## ⚠️ 常见问题

### Q1: 我不小心点了重置，树枯萎了怎么办？
**A:** 枯萎的树也是学习历程的一部分，不能删除。把它当作提醒，下次更加专注！

### Q2: 可以暂停吗？
**A:** 可以！点击暂停不算放弃，继续后树苗会继续成长。

### Q3: 数据会丢失吗？
**A:** 数据保存在浏览器本地，除非清除缓存，否则永久保存。

### Q4: 可以修改已种的树吗？
**A:** 不可以。这是为了保持记录的真实性。

### Q5: 森林可以导出吗？
**A:** 当前版本暂不支持，未来会添加。

### Q6: 可以自定义树木图标吗？
**A:** 当前版本不支持，未来可能会添加。

### Q7: 有没有成就系统？
**A:** 有！点击 🏆 成就按钮查看。完成番茄钟会获得经验和成就。

### Q8: 手机上可以用吗？
**A:** 完全支持！界面会自动适配手机屏幕。

## 🔄 数据管理

### 备份数据
```javascript
// 打开浏览器控制台（F12）
// 复制以下内容到文件保存
console.log(localStorage.getItem('forestData'))
```

### 恢复数据
```javascript
// 打开浏览器控制台（F12）
// 粘贴之前备份的数据
localStorage.setItem('forestData', '{"trees":[...]}')
// 刷新页面
location.reload()
```

### 清空数据
```javascript
// 警告：此操作不可恢复！
// 打开浏览器控制台（F12）
localStorage.removeItem('forestData')
location.reload()
```
