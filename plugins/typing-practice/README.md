# 打字练习插件 ⌨️

## 功能介绍

基于 [TypeWords](https://github.com/zyronon/TypeWords) 理念开发的英语单词打字练习插件，通过键盘输入的方式帮助记忆单词。

## 核心功能

### 📚 多词库支持
- **基础词汇**：适合初学者，包含常用日常单词
- **大学英语四级**：CET-4 核心词汇
- **大学英语六级**：CET-6 进阶词汇

### 🎯 智能练习
- **实时反馈**：输入时立即提示对错，绿色表示正确，红色表示错误
- **发音朗读**：显示每个单词时自动朗读（使用浏览器 TTS）
- **音标显示**：标准音标帮助正确发音
- **中文释义**：快速理解单词含义

### 📊 统计分析
- **累计练习**：记录总练习单词数
- **正确次数**：统计正确输入的单词数
- **最快速度**：WPM (Words Per Minute) 速度统计
- **准确率**：单次练习的准确率计算

### 🎨 界面特色
- 渐变色彩设计，视觉舒适
- 实时进度显示
- 完成后详细结果统计
- 响应式设计，支持移动端

## 使用方法

1. **开启插件**
   - 点击底部工具栏的 "⌨️ 打字" 图标
   - 或在插件库中启用"打字练习"插件

2. **选择词库**
   - 在下拉菜单中选择想要练习的词库
   - 点击"开始练习"按钮

3. **开始练习**
   - 屏幕会显示一个英文单词及其音标和中文释义
   - 在输入框中输入该单词
   - 输入正确后自动进入下一个单词
   - 输入错误会提示正确答案，可重新输入

4. **查看结果**
   - 完成所有单词后会显示统计结果
   - 包括完成单词数、准确率、速度和用时

## 快捷键

- **回车键**：提交当前输入
- **ESC**：退出练习（需确认）

## 技术特性

- 纯前端实现，无需服务器
- 本地存储统计数据
- 支持云端同步（如果启用）
- 使用 Web Speech API 实现语音朗读
- 随机打乱单词顺序，避免记忆顺序

## 数据存储

所有练习统计数据保存在本地 `localStorage` 中：
- `typingPracticeStats`：包含累计统计数据

如果启用了云端同步功能，数据会自动同步到云端。
