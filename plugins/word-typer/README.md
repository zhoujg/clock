# 单词打字背诵插件 📖

通过打字的方式背单词，结合英语学习和打字练习，参考 [qwerty-learner](https://github.com/RealKai42/qwerty-learner) 设计理念。

### 🎯 学习模式
1. **学习新词** - 学习未掌握或熟练度低的单词
2. **复习错词** - 重点复习曾经打错的单词
3. **全部复习** - 完整复习选定词库

### 💡 核心功能
- ✅ **逐字母实时验证** - 类似 qwerty-learner，即时反馈
- ✅ **视觉反馈** - 正确字母显示绿色，错误显示红色，当前待输入显示蓝色
- ✅ **虚拟键盘** 🆕 - 键盘练习模式显示虚拟键盘，实时高亮下一个要按的键
- ✅ **完整单词信息** - 显示音标、翻译、例句
- ✅ **优质发音** 🆕 - 智能选择最佳语音（Samantha、Alex 等），音质优秀
- ✅ **语音开关** 🆕 - 一键开启/关闭发音，状态持久化
- ✅ **智能记忆** - 记录每个单词的正确/错误次数
- ✅ **熟悉度评分** - 0-5 级熟悉度系统
- ✅ **学习统计** - 累计学习、熟练单词、准确率统计
- ✅ **无缝键盘输入** - 无需输入框，直接键盘输入
- ✅ **自动提交** - 输入完成自动检查，无需按回车

### 🎨 设计风格
- **Apple 毛玻璃效果** - backdrop-filter 磨砂玻璃
- **流畅动画** - 脉冲、弹跳、震动等动画效果
- **渐变配色** - 蓝紫色主题渐变
- **响应式布局** - 完美支持移动端
- **深色模式** - 自动适配系统深色模式

## 📖 使用方法

### 开始学习

1. 点击底部工具栏的 **"背单词"** 按钮
2. 选择词库：
   - ⌨️ **键盘练习**（初学者）🆕 - 适合打字初学者
   - 📘 **大学英语四级** - 适合四级备考
   - 📙 **大学英语六级** - 适合六级备考
   - 其他词库...
3. 选择学习模式：
   - 📚 **学习新词** - 学习未熟练的单词
   - 🔄 **复习错词** - 复习曾经打错的单词
   - 📝 **全部复习** - 完整复习词库
4. 根据显示的单词，直接在键盘上输入字母
5. 输入完成后自动检查并进入下一个单词

### 切换语音开关 🆕

1. 打开单词打字背诵面板
2. 点击词库选择器右侧的 **🔊** 按钮
3. 按钮变为 🔇 表示已关闭发音
4. 再次点击恢复发音
5. 设置自动保存，刷新页面后保持

### 键盘练习模式 🆕

1. 选择"**⌨️ 键盘练习（初学者）**"
2. 虚拟键盘会自动显示
3. **蓝色脉冲**高亮显示下一个要按的键
4. 按下正确的键后会显示**绿色**
5. **ASDF JKL** 基准键位有特殊标记

### 学习设置

- ☑️ **显示音标** - 显示/隐藏音标
- ☑️ **显示释义** - 显示/隐藏中文翻译
- ☑️ **自动发音** - 自动朗读单词

### 键盘操作

- **字母键（a-z）** - 输入字母
- **Backspace** - 删除最后一个字母
- **Escape** - 退出学习

### 其他功能

- **🔊 发音** - 手动播放单词发音
- **⏭️ 跳过** - 跳过当前单词
- **❌ 退出** - 退出学习，查看本次统计

## 🧠 学习原理

### 熟悉度系统

每个单词都有 0-5 级熟悉度评分：

- **0-1 级** - 陌生/初识
- **2-3 级** - 基本掌握
- **4-5 级** - 完全熟练

### 智能复习算法

- 打对一次 → 熟悉度 +1
- 打错一次 → 熟悉度 -1
- "学习新词"优先推送熟悉度 < 3 的单词
- "复习错词"推送错误次数 > 0 且熟悉度 < 4 的单词

### 学习记录

系统自动记录：
- 每个单词的正确/错误次数
- 最后学习时间
- 熟悉度级别
- 本次学习统计（正确、错误、准确率、用时）

## 🛠️ 技术实现

### 核心类：WordTyperManager

```javascript
// 主要方法
- init()                      // 初始化
- createUI()                  // 创建界面
- startLearning(mode)         // 开始学习（new/wrong/all）
- showNextWord()              // 显示下一个单词
- updateTypingDisplay()       // 更新打字显示（逐字母高亮）
- handleLetterInput(key)      // 处理字母输入
- handleCorrect()             // 处理正确答案
- handleWrong()               // 处理错误答案
- updateWordProgress(word, isCorrect)  // 更新单词进度
- pronounceWord()             // 发音
- updateStatsDisplay()        // 更新统计显示
```

### 数据结构

```javascript
// 词库结构
WORD_DICTIONARIES = {
    cet4: {
        name: '大学英语四级',
        icon: '📘',
        description: 'CET-4 核心词汇 2500+',
        color: '#007AFF',
        words: [...]
    }
}

// 单词结构
Word = {
    word: 'abandon',
    phonetic: '/əˈbændən/',
    translation: 'v. 放弃，抛弃',
    sentence: 'Never abandon your dreams.',
    sentenceTrans: '永远不要放弃梦想。'
}

// 学习进度
wordProgress = {
    'abandon': {
        correct: 5,       // 正确次数
        wrong: 1,         // 错误次数
        lastTime: 1234,   // 最后学习时间
        familiarity: 4    // 熟悉度 (0-5)
    }
}
```

### 本地存储

- `wordTyperSettings` - 用户设置（显示选项、发音开关等）
- `wordTyperProgress` - 学习进度（每个单词的记录）

### 样式特点

- **逐字母高亮**
  - `.word-letter.current` - 当前待输入（蓝色 + 脉冲动画）
  - `.word-letter.correct` - 已正确输入（绿色 + 弹跳动画）
  - `.word-letter.wrong` - 错误输入（红色 + 震动动画）

- **Apple 风格**
  - 毛玻璃背景：`backdrop-filter: blur(40px) saturate(180%)`
  - 圆角：24px（面板）、12px（按钮）
  - 阴影：`box-shadow: 0 20px 60px rgba(0, 0, 0, 0.15)`
  - 渐变：蓝紫色主题 (#007AFF → #5856D6)
