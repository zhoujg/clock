## 🔧 快速调试命令

如果遇到问题，在浏览器控制台执行以下命令：

### 检查音乐列表
```javascript
console.log('音乐数量:', window.app?.bgmPlayerManager?.musicList?.length);
console.log('当前播放:', window.app?.bgmPlayerManager?.currentTrack);
console.log('是否播放中:', window.app?.bgmPlayerManager?.isPlaying);
```

### 检查歌词显示
```javascript
console.log('歌词元素:', document.getElementById('lyricsDisplay'));
console.log('歌词显示状态:', document.getElementById('lyricsDisplay')?.style.display);
console.log('励志语元素:', document.querySelector('.quote-container'));
console.log('励志语显示状态:', document.querySelector('.quote-container')?.style.display);
```

### 手动触发歌词更新
```javascript
window.app?.updateLyricsDisplay();
```

### 强制显示歌词（测试用）
```javascript
document.getElementById('lyricsDisplay').style.display = 'block';
document.querySelector('.quote-container').style.display = 'none';
```

### 重新加载音乐
```javascript
await window.app?.bgmPlayerManager?.loadJamendoMusic({tags: 'ambient+instrumental', limit: 20});
```
