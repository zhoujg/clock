# Picsum 随机图片功能 - 快速开始

## 🚀 快速开始

### 第一步：打开应用
在浏览器中打开 `index.html` 文件

### 第二步：打开设置面板
点击右上角的设置按钮（⚙️齿轮图标）

### 第三步：使用随机图片
在设置面板中，你会看到以下新增选项：

1. **🎲 随机图片** - 点击加载一张随机背景图
2. **☆ 收藏图片** - 收藏当前显示的图片
3. **📁 图片收藏夹** - 管理你收藏的图片

## 📖 使用示例

### 示例 1：加载第一张随机图片
```
1. 点击设置按钮 ⚙️
2. 点击 "🎲 随机图片"
3. 等待 2-3 秒，背景图片会自动更新
4. 如果喜欢这张图片，点击 "☆ 收藏图片"
```

### 示例 2：浏览和使用收藏
```
1. 点击设置按钮 ⚙️
2. 点击 "📁 图片收藏夹"
3. 在收藏夹中看到所有收藏的图片缩略图
4. 点击图片下方的 ✓ 按钮应用该图片
5. 点击图片下方的 🗑️ 按钮删除收藏
```

### 示例 3：持续寻找完美背景
```
1. 多次点击 "🎲 随机图片" 浏览不同图片
2. 遇到喜欢的就点击 "☆ 收藏图片"
3. 可以收藏多张图片
4. 最后从收藏夹中选择最喜欢的一张
```

## 💡 使用技巧

### 技巧 1：快速切换
- 收藏夹中的图片可以快速切换，无需重新加载

### 技巧 2：收藏管理
- 收藏按钮会显示当前状态：
  - ☆ 空心星星 = 未收藏
  - ⭐ 实心星星 = 已收藏

### 技巧 3：窗口大小调整
- 调整浏览器窗口大小后，点击随机图片会加载适应新尺寸的图片

### 技巧 4：收藏夹位置
- 桌面端：收藏夹面板显示在右上角
- 手机端：收藏夹面板自动适应屏幕宽度

## 🎨 功能亮点

✅ **自动适配** - 根据屏幕尺寸自动加载合适大小的图片  
✅ **无需注册** - 使用免费的 Lorem Picsum API  
✅ **快速加载** - 图片加载速度快，体验流畅  
✅ **本地存储** - 收藏的图片信息保存在浏览器中  
✅ **响应式设计** - 完美支持各种设备和屏幕尺寸  

## ⚠️ 注意事项

1. **需要网络连接** - 加载随机图片需要互联网连接
2. **首次加载** - 第一次加载可能需要几秒钟
3. **收藏限制** - 建议收藏不超过 50 张图片，以免占用过多存储空间

## 🐛 常见问题

### Q: 点击随机图片后没有反应？
A: 请检查网络连接，并稍等几秒钟让图片加载

### Q: 收藏的图片会丢失吗？
A: 不会，收藏信息保存在浏览器的 LocalStorage 中，除非清除浏览器数据

### Q: 可以指定图片分类吗？
A: 当前版本是完全随机的，后续版本会添加分类功能

### Q: 图片质量如何？
A: Lorem Picsum 提供的都是高质量的摄影作品
