# BGM 音乐播放器使用指南

## 功能概述

BGM 音乐播放器是一个内置的背景音乐播放功能，让你在使用时钟应用时可以播放舒缓的背景音乐，提升专注度和氛围。

## 主要功能

### 1. 音乐播放控制
- ▶️ **播放/暂停**: 点击中央的播放按钮
- ⏮️ **上一曲**: 播放列表中的上一首音乐
- ⏭️ **下一曲**: 播放列表中的下一首音乐
- 🔁 **单曲循环**: 循环播放当前音乐

### 2. 播放列表
- 从 `assets/bgm` 目录自动加载音乐文件
- 点击任意曲目即可播放
- 当前播放的音乐会高亮显示

### 3. 进度控制
- 实时显示播放进度
- 点击进度条可快速跳转
- 显示当前时间和总时长

### 4. 音量调节
- 使用滑块调节音量（0-100%）
- 默认音量为 50%
- 设置会自动保存

### 5. 界面控制
- 🎵 **展开/收起**: 点击右下角的音乐图标
- ⬆️ **收起面板**: 点击面板右上角的向上箭头
- 收起后只显示音乐图标，节省屏幕空间

## 使用步骤

### 步骤 1: 添加音乐文件

1. 准备音乐文件（支持 MP3、WAV、OGG、M4A 格式）
2. 将音乐文件复制到 `assets/bgm/` 目录
3. 编辑 `scripts/bgmPlayer.js` 文件

### 步骤 2: 配置音乐列表

在 `scripts/bgmPlayer.js` 中找到 `loadMusicList()` 方法，添加你的音乐：

```javascript
async loadMusicList() {
    this.musicList = [
        { name: '轻音乐1', file: 'assets/bgm/music1.mp3' },
        { name: '轻音乐2', file: 'assets/bgm/music2.mp3' },
        { name: '自然之声', file: 'assets/bgm/nature.mp3' },
        { name: '钢琴曲', file: 'assets/bgm/piano.mp3' },
        { name: '专注白噪音', file: 'assets/bgm/whitenoise.mp3' }
    ];
    
    this.renderMusicList();
}
```

### 步骤 3: 开始播放

1. 打开应用，点击右下角的音乐图标展开播放器
2. 从音乐列表中选择一首音乐
3. 或直接点击播放按钮播放第一首音乐
4. 使用控制按钮切换歌曲、调节音量等

## 设置保存

播放器会自动保存以下设置：

- ✅ 当前播放的曲目索引
- ✅ 音量大小
- ✅ 是否开启单曲循环

下次打开应用时会自动恢复这些设置（不会自动播放）。

## 界面位置

- **桌面版**: 固定在屏幕右下角
- **移动版**: 固定在屏幕右下角，宽度自适应
- **收起后**: 只显示音乐图标按钮

## 推荐音乐类型

适合专注工作/学习的音乐：

- 🎹 **轻音乐/纯音乐**: 无歌词，不会分散注意力
- 🎼 **古典音乐**: 巴赫、莫扎特、德彪西等
- 🎵 **Lo-fi 音乐**: 轻松舒缓的节奏
- 🌊 **自然音效**: 雨声、森林、海浪、溪流
- 📻 **白噪音**: 咖啡厅环境音、粉红噪音

## 技术说明

### 文件结构

```
clock/
├── assets/
│   └── bgm/                    # 音乐文件目录
│       ├── README.md          # 目录说明
│       ├── music1.mp3
│       ├── music2.mp3
│       └── ...
├── scripts/
│   └── bgmPlayer.js           # 播放器逻辑
└── styles/
    └── bgmPlayer.css          # 播放器样式
```

### 支持的音频格式

- **MP3** (.mp3) - 推荐，兼容性最好
- **WAV** (.wav) - 无损音质，文件较大
- **OGG** (.ogg) - 开源格式，压缩率高
- **M4A** (.m4a) - Apple 格式，音质好

### 浏览器兼容性

- ✅ Chrome/Edge (推荐)
- ✅ Firefox
- ✅ Safari
- ✅ 移动端浏览器

### 自动播放限制

由于浏览器的自动播放策略，音乐不会在页面加载时自动播放。用户需要手动点击播放按钮。

## Android 应用支持

如果你在构建 Android 应用，音乐文件也需要放在：

```
android/app/src/main/assets/public/assets/bgm/
```

构建脚本会自动同步这些文件。

## 故障排除

### 音乐无法播放

1. **检查文件路径**: 确保音乐文件在 `assets/bgm/` 目录
2. **检查文件格式**: 确保是支持的音频格式
3. **检查配置**: 确保在 `bgmPlayer.js` 中正确配置了文件名
4. **检查控制台**: 打开浏览器开发者工具查看错误信息

### 音乐列表为空

- 确保已在 `bgmPlayer.js` 的 `loadMusicList()` 方法中添加音乐配置
- 刷新页面重新加载

### 音量太小或太大

- 使用播放器中的音量滑块调节
- 也可以在 `bgmPlayer.js` 中修改默认音量（0.0-1.0）

```javascript
this.volume = 0.5; // 默认音量 50%
```
