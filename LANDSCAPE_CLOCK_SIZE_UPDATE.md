# 横屏时钟大小调整

## 修改内容

将手机横屏模式下的翻转时钟大小放大 **1.25倍**。

## 修改的文件

### `styles/main.css`

#### 1. 常规横屏设备 (max-width: 900px, max-height: 500px)

**修改前**:
```css
.tick {
    font-size: 16vh;
    padding-bottom: 0.5em;
}
```

**修改后**:
```css
.tick {
    font-size: 20vh;  /* 16vh × 1.25 = 20vh */
    padding-bottom: 0.5em;
}
```

#### 2. 更小的横屏设备 (max-width: 700px, max-height: 400px)

**修改前**:
```css
.tick {
    font-size: 14vh;
}
```

**修改后**:
```css
.tick {
    font-size: 17.5vh;  /* 14vh × 1.25 = 17.5vh */
}
```

## 技术细节

### 使用 vh 单位的原因
- `vh` (viewport height) = 视口高度的百分比
- 横屏时高度是限制因素
- 使用 vh 确保时钟大小随屏幕高度自适应
- `20vh` = 屏幕高度的 20%

### 媒体查询断点
```css
/* 常规横屏 */
@media (max-width: 900px) and (max-height: 500px) and (orientation: landscape)

/* 更小横屏 */
@media (max-width: 700px) and (max-height: 400px) and (orientation: landscape)
```

### 计算示例
假设屏幕高度为 400px：
- 之前：16vh = 400 × 0.16 = 64px
- 现在：20vh = 400 × 0.20 = 80px
- 增加：80 - 64 = 16px (+25%)

## 回滚方法

如果觉得太大，可以调整倍数：

### 恢复原始大小 (1倍)
```css
/* 常规横屏 */
.tick {
    font-size: 16vh;
}

/* 小横屏 */
.tick {
    font-size: 14vh;
}
```

### 使用其他倍数

**1.1倍** (较小增幅):
```css
/* 常规横屏 */
.tick {
    font-size: 17.6vh;  /* 16 × 1.1 */
}

/* 小横屏 */
.tick {
    font-size: 15.4vh;  /* 14 × 1.1 */
}
```

**1.5倍** (更大):
```css
/* 常规横屏 */
.tick {
    font-size: 24vh;  /* 16 × 1.5 */
}

/* 小横屏 */
.tick {
    font-size: 21vh;  /* 14 × 1.5 */
}
```
