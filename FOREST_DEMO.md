# 🌲 虚拟森林系统 - 功能演示

## 📹 演示流程

### 场景 1：成功种植一棵树

```
1. 用户打开时钟应用
2. 点击右上角 🍅 番茄钟按钮
3. 选择「工作」模式（25分钟）
4. 点击播放按钮 ▶️

→ 结果：
- 倒计时开始
- 左上角出现 🌲 森林按钮
- 点击森林按钮，可以看到小树苗正在成长
- 进度条实时更新
- 树苗图标从 🌱 逐渐变化

5. 等待 25 分钟倒计时完成（或快进测试）

→ 结果：
- 🎉 弹出成功通知："获得一棵小树"
- 森林徽章数字 +1
- 打开森林面板，可以看到新种的树
```

### 场景 2：中途放弃导致枯萎

```
1. 开始一个番茄钟
2. 运行一段时间后
3. 点击「重置」按钮

→ 结果：
- 💔 弹出失败通知："树苗枯萎了"
- 森林中记录一棵枯萎的树 🥀
- 不影响健康树木计数
- 但会降低成功率
```

### 场景 3：暂停不算放弃

```
1. 开始一个番茄钟
2. 运行一段时间后
3. 点击「暂停」按钮
4. 再次点击「播放」继续

→ 结果：
- 树苗继续成长
- 进度保持
- 没有任何惩罚
```

### 场景 4：查看完整森林

```
1. 种植几棵树后
2. 点击 🌲 森林按钮
3. 点击「查看完整森林」按钮

→ 结果：
- 新标签页打开 forest.html
- 显示所有种植的树木
- 统计数据一目了然
- 悬停树木有动画效果
- 背景有浮动的叶子粒子
```

## 🎯 用户体验流程

### 首次使用
```
1. 新用户看到森林按钮，但徽章为空
2. 点击查看，面板显示"还没有种植任何树木"
3. 提示用户去完成一个番茄钟
4. 完成后，获得第一棵树，产生成就感
```

### 日常使用
```
1. 每天打开应用
2. 查看森林徽章数字，了解昨天的成果
3. 开始今天的专注
4. 看着徽章数字增长
5. 晚上打开森林页面，欣赏今天的成果
```

### 长期使用
```
1. 一周后，森林已经初具规模
2. 一月后，森林茂密
3. 查看成功率，了解自己的专注能力
4. 设定目标：提高成功率、种植更大的树
```
