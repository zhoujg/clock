# 🌲 虚拟森林系统 - Forest 功能

## 📖 功能介绍

灵感来源于 **Forest** 应用，让你的每一次专注学习都化作一棵生机勃勃的树木！

### 核心机制

- **🌱 种树机制**：每开始一个番茄钟或自定义倒计时，就开始种下一棵树苗
- **🌳 成长过程**：专注完成整个计时，树木健康成长
- **🥀 枯萎惩罚**：中途放弃（点击重置），树苗会枯萎
- **🌲 森林记录**：用郁郁葱葱的森林，直观展示你的所有努力

## ✨ 功能特点

### 1. 树木类型

根据专注时长，可以种植不同类型的树木：

| 时长 | 树木类型 | 图标 |
|------|---------|------|
| 15分钟 | 小树苗 | 🌱 |
| 25分钟 | 小树 | 🌿 |
| 45分钟 | 大树 | 🌳 |
| 60分钟+ | 参天大树 | 🌲 |

### 2. 实时成长显示

在森林面板中可以看到：
- 🌱 当前正在成长的树苗
- ⏱️ 剩余时间倒计时
- 📊 成长进度条
- 🎨 树苗随着进度逐渐成长的动画

### 3. 统计数据

森林面板显示完整的统计信息：
- 📈 总种植树木数量
- ✅ 健康树木数量
- ❌ 枯萎树木数量
- 📜 最近10次种植记录

### 4. 完整森林视图

点击「查看完整森林」按钮，打开独立的森林页面：
- 🌲 以网格形式展示所有树木
- 📊 森林统计数据（总数、健康、枯萎、成功率）
- 🎨 精美的森林主题背景
- 🍃 动态浮动的叶子粒子效果
- 📱 响应式设计，完美适配各种屏幕

## 🎮 使用方法

### 开始种树

1. 点击右上角的 **🍅 番茄钟按钮**
2. 选择工作模式（25分钟）或自定义时长
3. 点击 **播放按钮** 开始倒计时
4. 🌱 **树苗开始成长！**

### 观察成长

在倒计时过程中：
- 点击 **🌲 森林按钮** 查看当前树苗
- 可以看到实时的成长进度
- 树苗图标会随进度变化：🌱 → 🌿 → 🌳 → 🌲

### 完成种植

专注完成整个倒计时：
- ✅ 树木健康成长
- 🎉 获得成功种植通知
- 🏆 树木被添加到你的森林

### 放弃的后果

如果中途点击「重置」按钮：
- 💔 树苗会枯萎
- 🥀 枯萎的树也会被记录
- 📉 影响森林成功率

> **💡 提示**：暂停不算放弃！可以放心暂停休息，继续后树苗会继续成长。

## 💾 数据存储

所有森林数据保存在浏览器本地存储中：

```javascript
{
  trees: [
    {
      type: 'medium',        // 树木类型
      startTime: 1234567890, // 开始时间
      duration: 1500000,     // 时长（毫秒）
      status: 'completed',   // 状态：completed/dead
      completedTime: 1234569390 // 完成时间
    },
    // ... 更多树木
  ]
}
```

数据永久保存，除非清除浏览器缓存。

## 🔧 技术实现

### 集成方式
- `scripts/forest.js` - 森林系统核心逻辑
- `styles/forest.css` - 森林界面样式
- `forest.html` - 独立森林展示页面

### 与番茄钟集成
- 番茄钟开始 → `forestSystem.startPlanting()`
- 每秒更新 → `forestSystem.updateTreeProgress()`
- 成功完成 → `forestSystem.completePlanting()`
- 中途重置 → `forestSystem.abandonPlanting()`

### 与成就系统配合
- 森林系统和成就系统共享番茄钟完成事件
- 未来可以添加森林相关成就（例如：种植100棵树）
