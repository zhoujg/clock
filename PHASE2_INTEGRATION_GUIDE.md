# Phase 2 整合功能 - 使用指南 🚀

## 📋 概述

Phase 2 将"每日三个故事"与番茄钟、成就系统、森林系统深度整合，打造完整的目标-执行-奖励闭环。

## 🔗 三大整合功能

### 1. 番茄钟整合 🍅

#### 功能说明
- 为每个故事关联番茄钟
- 自动追踪每个故事的时间投入
- 完成番茄钟后自动更新故事数据

#### 使用方法

**步骤 1：设定故事**
```
故事标题：完成项目文档
故事描述：今天要完成项目文档的第3章...
价值观：💼 责任
```

**步骤 2：关联番茄钟**
- 填写故事标题后，会出现"🍅 开始番茄钟"按钮
- 点击按钮，自动关联并打开番茄钟面板
- 系统会显示关联提示通知

**步骤 3：开始工作**
- 在番茄钟面板选择时长（25分钟/自定义）
- 点击开始，专注完成故事任务
- 番茄钟计时的同时，森林系统也在种树

**步骤 4：完成反馈**
- 完成番茄钟后，系统自动记录：
  - ✅ 番茄钟数量 +1
  - ⏱️ 时间投入更新
  - 📊 统计数据更新

#### 数据追踪

每个故事卡片会显示：
```
🍅 3个          - 已完成3个番茄钟
⏱️ 75分钟       - 总投入时间
```

### 2. 成就系统整合 🏆

#### 新增成就

**故事相关成就**（5个）

| 成就 | 图标 | 要求 | 奖励 |
|------|------|------|------|
| 故事开端 | 📖 | 完成第一个故事 | 经验值 +2 |
| 完美一天 | 🎯 | 完成当天所有三个故事 | 经验值 +10 |
| 故事之周 | 📚 | 连续7天完成所有故事 | 经验值 +50 |
| 故事大师 | 🏆 | 累计完成30个故事 | 经验值 +60 |
| 故事传奇 | 👑 | 累计完成100个故事 | 经验值 +200 |

#### 成就触发机制

1. **自动检测**
   - 每次完成故事时自动检查
   - 符合条件立即解锁成就
   - 弹出成就解锁动画

2. **经验奖励**
   - 完成单个故事：+30 经验
   - 解锁成就：额外经验值
   - 经验累积自动升级

3. **查看成就**
   - 点击右上角 🏆 成就按钮
   - 选择"故事"分类标签
   - 查看解锁进度和完成情况

#### 成就面板新增

在成就模态框中新增"故事"分类：
```
🎯 全部  |  🍅 番茄钟  |  ⏰ 时长  |  🔥 连续  |  📖 故事  |  ⭐ 特殊
```

## 💡 使用技巧

### 技巧 1：合理分配番茄钟

```
大任务（2小时+）：
- 故事1：完成项目设计（💼 责任）
  → 使用4-5个番茄钟，分段完成

中等任务（1小时）：
- 故事2：学习新知识（📚 学习）
  → 使用2-3个番茄钟

小任务（30分钟）：
- 故事3：整理笔记（⚡ 效率）
  → 使用1-2个番茄钟
```

### 技巧 2：价值观平衡

查看统计面板的价值观分布：
```
如果发现：
- 💼 责任 占70% → 工作太多，注意平衡
- 💪 健康 占5%  → 需要增加运动时间
- 👨‍👩‍👧 家庭 占10% → 增加家人陪伴
```

### 技巧 3：成就导向

设定短期目标：
```
本周目标：
- [ ] 连续7天完成所有故事 → 解锁"故事之周"
- [ ] 累计完成20个故事 → 距离"故事大师"还差10个
- [ ] 种植50棵健康树木 → 建造郁郁葱葱的森林
```

### 技巧 4：番茄钟策略

```
工作任务：
- 25分钟工作 + 5分钟短休息
- 重复4次后，15分钟长休息

学习任务：
- 45分钟深度学习 + 10分钟休息
- 适合需要长时间集中的内容

创意任务：
- 15分钟快速迭代 + 5分钟休息
- 保持思维活跃，避免僵化
```

## 🐛 常见问题

### Q1: 番茄钟中途取消，故事数据会丢失吗？
**A**: 不会。只有完成的番茄钟才会计入故事数据。中途取消不影响已有数据。

### Q2: 可以一个故事关联多个番茄钟吗？
**A**: 可以。每次完成番茄钟后，可以继续点击"开始番茄钟"继续累积。

### Q3: 完成故事但没用番茄钟，会有奖励吗？
**A**: 会。完成故事仍然获得：
- 经验值 +30
- 完成动画
- 成就检查（如"完美一天"）
- 统计数据更新

### Q4: 如何查看历史的故事和番茄钟数据？
**A**: 
1. 打开每日三个故事面板
2. 点击"查看统计"
3. 查看最近7天趋势和价值观分布

### Q5: 成就解锁后能重复获得吗？
**A**: 大部分成就只能解锁一次。但"每日目标"类成就（如"今日学霸"）可以重复获得。
