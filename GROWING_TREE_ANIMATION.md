# 🌱 首页成长树苗动画功能

## 🎨 功能说明

在番茄钟倒计时时，首页时钟旁边会显示一棵**实时成长的树苗**，随着时间推移不断长大，直观展示专注进度。

## 🎬 使用流程

### 1. 开始种树
```javascript
用户点击播放 → 番茄钟开始
↓
forestSystem.startPlanting(duration)
↓
树苗显示在首页
- 淡入效果（0.5秒）
- 初始图标：🌱
- 进度：0%
```

### 2. 实时成长
```javascript
每秒更新一次
↓
forestSystem.updateTreeProgress(remainingSeconds)
↓
更新树苗状态
- 计算进度百分比
- 更新进度条宽度
- 根据进度切换图标
- 调整树苗大小
```

### 3. 完成种植
```javascript
倒计时结束
↓
forestSystem.completePlanting()
↓
播放完成动画
- 树苗欢快跳跃（1秒）
- 淡出消失（0.5秒）
- 显示成功通知
```

### 4. 中途放弃
```javascript
用户点击重置
↓
forestSystem.abandonPlanting()
↓
播放枯萎动画
- 树苗变成 🥀
- 缩小旋转灰化（0.8秒）
- 淡出消失（0.5秒）
- 显示失败通知
```

## 📊 进度计算

### 计算公式
```javascript
// 总时长（秒）
const totalSeconds = duration * 60

// 已用时间（秒）
const elapsedSeconds = totalSeconds - remainingSeconds

// 进度百分比
const progress = (elapsedSeconds / totalSeconds) * 100

// 示例：25分钟番茄钟，已过去10分钟
// totalSeconds = 25 * 60 = 1500
// elapsedSeconds = 10 * 60 = 600
// progress = (600 / 1500) * 100 = 40%
```

### 图标切换逻辑
```javascript
if (progress < 20) {
    icon = '🌱'  // 种子期
    size = 60px
} else if (progress < 40) {
    icon = '🌿'  // 发芽期
    size = 70px
} else if (progress < 60) {
    icon = '🌳'  // 幼树期
    size = 80px
} else if (progress < 80) {
    icon = treeType.emoji  // 成熟期
    size = 90px
} else {
    icon = treeType.emoji  // 完全成长
    size = 100px
}
```
