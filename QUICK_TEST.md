## 常见问题

### Q: 看不到音乐列表？
A: 确保：
1. 使用 HTTP 服务器（不是 file://）
2. `assets/bgm/` 目录中有音乐文件
3. 打开浏览器控制台查看错误信息

### Q: 按钮太小看不清？
A: 这是预期的设计，按钮尺寸为原来的 50%。如果需要调整，可以修改 `styles/controls.css` 中的：
```css
.control-btn {
    width: 25px;   /* 可以调整为 30px 或 35px */
    height: 25px;
}
```

### Q: 音乐按钮和设置按钮离得太近？
A: 可以调整 `styles/controls.css` 中的：
```css
.controls {
    gap: 10px;  /* 可以调整为 15px 或 20px */
}
```

## 浏览器控制台检查

打开浏览器开发者工具（F12），查看：

1. **Console 标签**：
   - 应该看到 "从 music-list.json 加载了 X 首音乐" 或 "使用默认音乐列表"
   - 不应该有红色错误信息

2. **Network 标签**：
   - 检查 `music-list.json` 是否成功加载（或 404）
   - 检查音乐文件是否能正常加载

3. **Elements 标签**：
   - 查看 `.music-panel` 元素是否存在
   - 查看按钮尺寸是否正确
