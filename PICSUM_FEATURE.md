# Picsum 随机图片功能说明

## 功能概述

时钟应用现在支持使用 Lorem Picsum API 来加载随机的高质量背景图片，并提供收藏功能。

## 功能特点

### 1. 随机图片
- **自动适配**：根据当前窗口大小自动加载合适尺寸的图片
- **无需注册**：使用 Lorem Picsum 免费 API，无需任何注册和 API 密钥
- **稳定可靠**：使用成熟稳定的图片服务
- **一键刷新**：点击"随机图片"按钮即可加载新的背景图

### 2. 收藏功能
- **一键收藏**：喜欢的图片可以立即收藏
- **收藏夹管理**：查看所有收藏的图片
- **快速应用**：从收藏夹中快速选择并应用图片
- **删除收藏**：不喜欢的收藏可以随时删除

### 3. 持久化存储
- **自动保存**：当前使用的 Picsum 图片会自动保存
- **下次恢复**：重新打开应用时会恢复上次使用的图片
- **收藏同步**：收藏的图片列表会保存在本地存储中

## 使用方法

### 加载随机图片
1. 点击右上角的设置按钮（⚙️）
2. 在设置面板中点击"随机图片"（🎲）
3. 等待图片加载完成
4. 如果不满意，可以再次点击加载新图片

### 收藏当前图片
1. 加载一张随机图片后
2. 点击设置面板中的"收藏图片"（☆）
3. 图标会变成实心星星（⭐）表示已收藏
4. 再次点击可以取消收藏

### 管理收藏夹
1. 点击设置面板中的"图片收藏夹"（📁）
2. 打开收藏夹面板
3. 点击图片上的"✓"按钮应用该图片
4. 点击图片上的"🗑️"按钮删除收藏

## 技术实现

### API 说明
- **服务提供商**：Lorem Picsum (https://picsum.photos/)
- **图片格式**：JPEG
- **尺寸适配**：`https://picsum.photos/seed/{seed}/{width}/{height}`
- **随机性**：使用时间戳作为 seed 确保每次获取不同图片

### 存储方案
- **LocalStorage**：用于保存收藏列表和当前图片信息
- **存储键**：
  - `picsumFavorites`：收藏的图片列表
  - `flipClockSettings.picsumId`：当前图片 ID
  - `flipClockSettings.picsumUrl`：当前图片 URL

## 注意事项

1. **网络连接**：需要互联网连接才能加载图片
2. **加载时间**：首次加载可能需要几秒钟，取决于网络速度
3. **存储空间**：收藏的图片信息保存在 LocalStorage，占用空间很小
4. **图片质量**：Picsum 提供的都是高质量摄影作品
5. **随机性**：每次加载的图片都是随机的，无法指定特定图片
