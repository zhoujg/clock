# 时钟卡片智能颜色功能

## ✨ 功能说明

时钟卡片现在能根据首页背景色自动智能调整背景颜色和文字颜色，确保最佳的可读性和视觉效果。

## 🎨 工作原理

### 1. 纯色背景模式
当使用纯色背景时，系统会：
- 分析背景色的亮度值
- 浅色背景（亮度 > 128）→ 使用深色卡片（背景色 -40 亮度）
- 深色背景（亮度 ≤ 128）→ 使用稍亮的卡片（背景色 +30 亮度）

**处理逻辑**：`BackgroundManager.updateFlipPanelColor()`

### 2. 背景图片模式 ✨ 新功能
当使用背景图片时，系统会：
- 自动采样时钟卡片所在区域的背景图片亮度
- 分析图片中心区域（35%-65% 宽度，45%-60% 高度）的颜色
- 根据分析结果智能调整：

#### 深色背景图片
- 卡片背景：半透明浅色 `rgba(255, 255, 255, 0.15)`
- 数字颜色：高对比度白色 `rgba(255, 255, 255, 0.98)`
- 阴影效果：深色阴影 `rgba(0, 0, 0, 0.6)`
- 添加毛玻璃效果（backdrop-filter）

#### 浅色背景图片
- 卡片背景：半透明深色 `rgba(0, 0, 0, 0.18)`
- 数字颜色：高对比度白色 `rgba(255, 255, 255, 0.98)`
- 阴影效果：强深色阴影 `rgba(0, 0, 0, 0.8)`
- 添加毛玻璃效果（backdrop-filter）

**处理逻辑**：`SmartColorManager.adjustClockPanelColors()`

## 🔧 技术实现

### 采样区域配置
系统在以下三个关键区域采样：
```javascript
[
    { x: 0.25, y: 0.06, width: 0.15, height: 0.06 },  // 日期文字区域
    { x: 0.45, y: 0.06, width: 0.1, height: 0.06 },   // 星期文字区域
    { x: 0.35, y: 0.45, width: 0.3, height: 0.15 }    // 时钟卡片区域（中心位置）⭐ 新增
]
```

### 亮度计算公式
使用人眼感知亮度公式：
```javascript
brightness = (0.299 * R + 0.587 * G + 0.114 * B)
```

### 阈值设定
- **亮度阈值**：60%（低于60%判定为深色背景）
- 这个阈值更容易触发浅色文字模式，提供更好的可读性

## 🎯 触发时机

智能颜色调整会在以下情况自动触发：

1. **切换纯色背景**时（`BackgroundManager.setColor()`）
2. **上传/切换背景图片**时（`BackgroundManager.setImage()`）
3. **页面加载时**（延迟 500ms 后自动分析）
4. **背景变化时**（MutationObserver 监听 body 的 style 属性）
5. **手动刷新**（调用 `window.smartColorManager.refresh()`）

## 🛠️ 调试功能

### 开启调试模式
按下快捷键：`Ctrl + Shift + D`

调试模式会：
- 在屏幕上显示采样区域的可视化框框
- 在控制台输出详细的颜色分析信息
- 显示每个区域的亮度值和平均亮度
- 显示当前选择的颜色模式（深色/浅色）

## 🎨 视觉效果

### 毛玻璃效果
时钟卡片使用现代的毛玻璃效果（backdrop-filter），使数字更加清晰易读，同时保持与背景的和谐融合：
- 标准浏览器：`backdrop-filter: blur(10px)`
- Safari 浏览器：`-webkit-backdrop-filter: blur(10px)`

### 高对比度
无论背景图片是深色还是浅色，时钟数字始终保持：
- 高对比度白色文字
- 适当的阴影效果
- 半透明卡片背景，确保可读性

## 📝 代码位置

- **智能颜色管理器**：`scripts/smartColor.js`
  - `adjustClockPanelColors()` - 时钟卡片颜色调整 ⭐ 新增
  - `analyzeImageBrightness()` - 图片亮度分析
  - `adjustTextColor()` - 文字颜色调整

- **背景管理器**：`scripts/background.js`
  - `updateFlipPanelColor()` - 纯色背景下的卡片颜色
  - `setImage()` - 触发智能颜色分析 ⭐ 更新

- **应用控制器**：`scripts/app.js`
  - `loadSavedSettings()` - 加载设置时触发颜色分析

## 🎉 使用建议

1. **上传图片后**，系统会自动分析并调整所有元素的颜色
2. **切换不同图片**时，颜色会实时自动适配
3. **如果颜色不理想**，可以尝试其他图片或使用纯色背景
4. **开启调试模式**可以查看详细的颜色分析过程
