## 🎯 设计理念

### 视觉层级
```
1. 时钟（中心，最重要）
2. 励志语（下方，激励作用）
3. 树苗（右下角，辅助反馈）
```

### 注意力分配
```
主要注意力：时钟 + 励志语
外围视野：树苗成长
偶尔查看：树苗进度
```

### 空间利用
```
中心区域：核心功能（时钟、励志语）
边角区域：辅助功能（树苗）
上方区域：日期信息
```
